## Шаг 5. Работа с переменными и выводом

В ходе выполнения задания были созданы две переменные:

```php
<?php

$days = 288;
$message = "Все возвращаются на работу!";
```

Переменная `$days` является целочисленной и хранит значение 288.
Переменная `$message` является строковой и содержит текст: «Все возвращаются на работу!».

---

### Вывод с использованием конкатенации

Конкатенация — это объединение строк. В PHP для этого используется оператор `.`:

```php
echo $days . " " . $message;
```

В данном случае значения переменных объединяются в одну строку и выводятся на экран.

---

### Вывод с использованием двойных кавычек

При использовании двойных кавычек PHP автоматически подставляет значения переменных внутрь строки:

```php
echo "{$days} {$message}<br />";
```

Значения переменных выводятся без использования оператора конкатенации.

---

### Использование перехода на новую строку

Для переноса строки в браузере был использован HTML-тег:

```php
<br />
```

Это позволяет выводить данные с новой строки.

## Контрольные вопросы

### 1️. Какие способы установки PHP существуют?

Существует несколько способов установки PHP:

* Скачивание с официального сайта **php.net** и ручная установка.
* Установка через готовые сборки (XAMPP, OpenServer, WAMP).
* Установка через менеджеры пакетов (например, Chocolatey).
* Использование Docker-контейнера с PHP.

---

### 2️. Как проверить, что PHP установлен и работает?

Для проверки установки необходимо выполнить в командной строке команду:

```bash
php -v
```

Если отображается версия PHP, значит установка прошла успешно.

Также можно создать файл `index.php` с кодом:

```php
<?php echo "Test"; ?>
```

и запустить его через встроенный сервер PHP.

---

### 3️. Чем отличается оператор echo от print?

* `echo` используется для вывода текста и может выводить несколько значений.
* `print` выводит только одно значение.
* `echo` работает немного быстрее и чаще используется в практике.
