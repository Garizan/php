# Лабораторная работа 6: Управление рецептами

## Описание проекта
В рамках лабораторной работы был разработан веб-проект **«Управление рецептами»**, позволяющий пользователю:
- добавлять рецепты;
- сохранять их в файл;
- просматривать список рецептов;
- сортировать данные по различным полям.

Проект реализован с использованием **HTML, CSS и PHP**.

---

# Шаг 1. Определение модели данных

Для хранения рецептов используется следующая модель данных:

| Поле        | Тип       | Описание |
|------------|----------|----------|
| title       | string   | Название рецепта |
| ingredients | text     | Список ингредиентов |
| instructions| text     | Пошаговое описание |
| category    | enum     | Категория блюда (завтрак, обед, ужин, десерт) |
| prep_time   | number   | Время приготовления (в минутах) |
| difficulty  | enum     | Сложность (легко, средне, сложно) |
| created_at  | date     | Дата создания |
| author      | string   | Автор рецепта |


# Шаг 2. Создание HTML-формы

Была разработана HTML-форма для добавления рецепта.

### Основные особенности:
- Используется метод **POST**
- Все поля соответствуют модели данных
- Добавлена клиентская валидация:
  - `required`
  - `minlength`
  - `type="number"`
  - `type="date"`

### Возможности формы:
- ввод данных рецепта
- выбор категории и сложности
- отправка данных на сервер

---

# Шаг 3. Обработка данных на сервере (PHP)

Создан PHP-скрипт (`save.php`), который:

### Выполняет:
- получение данных из `$_POST`
- проверку обязательных полей
- проверку корректности даты
- формирование массива данных
- сохранение данных в файл `data.txt`

### Формат хранения:
Данные сохраняются в формате **JSON**:

```json
    {
        "title": "Суп",
        "ingredients": "Картофель, мясо",
        "instructions": "варим мясо\r\nварим картофель",
        "category": "обед",
        "prep_time": "40",
        "difficulty": "средне",
        "created_at": "2026-03-31",
        "author": "Луи"
    }
```

## Контрольные вопросы
1. Какие существуют методы отправки данных из формы на сервер? Какие методы поддерживает HTML-форма?
Основные методы отправки данных — GET и POST, и именно эти два метода поддерживаются HTML-формами.

2. Какие глобальные переменные используются для доступа к данным формы в PHP?
В PHP для доступа к данным формы используются суперглобальные массивы $_GET, $_POST и $_REQUEST.

3. Как обеспечить безопасность при обработке данных из формы (например, защититься от XSS)?
Для защиты от XSS необходимо экранировать пользовательский ввод с помощью функции htmlspecialchars() перед выводом в HTML.
