# 2. Условные конструкции

## Задание

Создать таблицу с расписанием сотрудников, формируемым на основе текущего дня недели с использованием функции `date()`.

### Условия:

* Для **John Styles**:

  * Если текущий день недели — понедельник, среда или пятница → `8:00–12:00`
  * В остальные дни → `Нерабочий день`

* Для **Jane Doe**:

  * Если текущий день недели — вторник, четверг или суббота → `12:00–16:00`
  * В остальные дни → `Нерабочий день`

---

## Программный код

```php
<?php
$day = (int)date('N');

$johnSchedule = '';
$janeSchedule = '';

// John Styles: Пн, Ср, Пт
if ($day === 1 || $day === 3 || $day === 5) {
    $johnSchedule = '8:00-12:00';
} else {
    $johnSchedule = 'Нерабочий день';
}

// Jane Doe: Вт, Чт, Сб
if ($day === 2 || $day === 4 || $day === 6) {
    $janeSchedule = '12:00-16:00';
} else {
    $janeSchedule = 'Нерабочий день';
}
?>
<table>
    <tr>
        <th>№</th>
        <th>Фамилия Имя</th>
        <th>График работы</th>
    </tr>
    <tr>
        <td>1</td>
        <td>John Styles</td>
        <td><?= $johnSchedule ?></td>
    </tr>
    <tr>
        <td>2</td>
        <td>Jane Doe</td>
        <td><?= $janeSchedule ?></td>
    </tr>
```

# 3. Циклы

## Исходный код

```php
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
   $a += 10;
   $b += 5;
}

echo "End of the loop: a = $a, b = $b";
```

## 3.1 Цикл for с выводом промежуточных значений

```php
<?php

$b = 0;

for ($i = 0; $i <= 5; $i++) {
   $a += 10;
   $b += 5;

   echo "Шаг $i: a = $a, b = $b <br>";
}

```

## 3.2 Переписывание цикла с использованием while

```php
<?php

$a = 0;
$b = 0;
$i = 0;

while ($i <= 5) {
   $a += 10;
   $b += 5;

   echo "Шаг $i: a = $a, b = $b <br>";
   $i++;
}

echo "End of the loop: a = $a, b = $b";
```

---

## 3.3 Переписывание цикла с использованием do-while

```php
<?php

$a = 0;
$b = 0;
$i = 0;

do {
   $a += 10;
   $b += 5;

   echo "Шаг $i: a = $a, b = $b <br>";
   $i++;
} while ($i <= 5);

echo "End of the loop: a = $a, b = $b";
```

# 4. Контрольные вопросы

## 4.1 В чем разница между циклами for, while и do-while?

**for**

* Используется, когда заранее известно количество повторений.
* Содержит счётчик, условие и шаг в одной строке.
* Удобен для работы с индексами.

**while**

* Используется, когда количество повторений заранее неизвестно.
* Выполняется, пока условие истинно.
* Проверка условия происходит перед выполнением тела цикла.

**do-while**

* Отличается тем, что сначала выполняет тело цикла, а затем проверяет условие.
* Гарантирует выполнение тела цикла минимум один раз.

---

## 4.2 Как работает тернарный оператор ? : в PHP?

Тернарный оператор — это сокращённая форма условного оператора `if-else`.

Синтаксис:

```php
(условие) ? значение_если_true : значение_если_false;
```

Пример:

```php
$age = 20;
$result = ($age >= 18) ? "Совершеннолетний" : "Несовершеннолетний";
```

Если условие истинно — возвращается первое значение, иначе второе.

---

## 4.3 Что произойдет, если в do-while поставить условие, которое изначально ложно?

Тело цикла выполнится **один раз**, так как проверка условия происходит **после выполнения** блока `do`.

Пример:

```php
$i = 0;

do {
    echo "Этот текст выведется один раз";
} while ($i > 10);
```

Даже если условие ложно, код внутри `do` выполнится один раз.
